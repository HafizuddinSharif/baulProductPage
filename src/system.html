Map generation using SLAM

Simultaneous localization and mapping (SLAM) allows us to map the playing area autonomously using LIDAR to feed the data for the robot

Steps:
1. Position robot in the middle of a room.
2. Robot uses random trees to identify points to visit.
3. Robot navigates to points it hasn't visited.
4. Prioritises areas it has not yet visited.

Algorithm used: Rapidly-exploring random tree (RRT) algorithm
This algorithm builds a tree of traces, where the tree is constructed by randomly generated traces incrementally in the search space starting at the robot initial position and is inherently biased to grow towards areas that have less information.

Movement of the robot with a path finding algorithm

Given a pre mapped room - identify areas to visit. Taking into account space between the wall and the areas that are physically available.

Algorithm used: Navigation stack - Dynamic Window Approach (DWA) algorithm

Simple heuristic algorithm that takes a (x,y) value as goal and outputs the velocity(v, w) and the trajectory to the goal. It searches through all the allowable velocity within the given range, do velocity dynamics and check for each velocity, will the distance to the nearest obstacle be within the robots braking distance. If the robot could stop before it , then mark the velocity as 'admissible'. After the iteration, calculate the optimal one using the internal cost function, and output the trajectory and the velocity to the move base.

Identification of table tennis balls

Thresholding: All pixels outside a colour threshold are removed. Allows us to isolate individual objects to an extent.

Image before preprocessing.

Image after thresholding.

Contours: Outline the isolated objects by highlighting the pixels next to the black background. These outlines tell us information about the object.

Image after contouring.

We can use several processes to tell us information about the object. Firstly we can check the size of the object is above a minimum. We can tell how curved the outline is (circles are very curved). We can also compare the width and height of the object. Circles have a a very similar width and height.

We then remove contours that don't meet this criteria.
